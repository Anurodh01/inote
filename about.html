ABOUT THE WEB


This is general purpose notes making web development and most importantly my first web development. In this a user can make their notes of daily use and doings by signup and login in this website. This website is created with the help of Html,Css,Javascript and Php and with the help of mysql database. In this i tried to give the functionalties of high security as well as i tried to use the password_hash() and password_verify() functions while storing the data and retrieving the data inside the database.In this web development all basic operations of the database as CRUD( create ,read and update and delete).IN the home page the notes can be added to the database and can be update and delete as well. In this web application, we can signup by entering the email and password, in this the unique email is taken. By entering the email and specified password in the login page we will welcome on the home page basic functionaltiesof adding note, editing notes and deleting notes are performed.
